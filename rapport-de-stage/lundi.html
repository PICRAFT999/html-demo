RETOUR

Dans l’entreprise, l’entrepôt est un endroit très important :
Ici, les tâches sont réparties en 10 postes et c’est donc ici que l’on stocke les vêtements.

La Création est le premier poste sur 10, chargé de mettre en stock les vêtements reçus.
Le Stockage range les vêtements dans le stock et enregistre leur emplacement.
Le Picking va chercher les vêtements.
Le Folding rassemble les mêmes vêtements, les plie et les met dans un colis.
Le Packing vérifie les colis et les ferme.
Le Sorting range les colis vers les bons transporteurs.
Puis, lorsque le temps de location d’un vêtement est terminé, il est renvoyé à l’entrepôt par un transporteur.
Le Returns ouvre les colis pour vérifier que tous les vêtements sont revenus en les scannant.
Le Washed lave les vêtements.
Le Hanging met les vêtements sur un cintre.
Le Inspected vérifie l’état des vêtements.
